AI Powered Autonomous Personal Cyber Defense System is a full-stack cybersecurity web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

This system provides real-time AI-based threat detection and automated protection against cyber attacks such as phishing attempts, malware behavior, suspicious login attempts, and abnormal activity patterns.

The system intelligently monitors user activity, analyzes logs, detects anomalies, and takes autonomous actions to protect the user.

-------------------------------------------------------------------------------------------

Key Features

' AI-based Threat Detection

' Real-time Security Dashboard

' JWT Authentication & Authorization

' Activity Log Monitoring

' Instant Threat Alerts & Notifications

' Automated Threat Blocking

' Threat History & Analytics

' Encrypted Data Storage

' Role-Based Access Control
